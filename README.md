# Abdalla Yaser - CyberCode Nexus Portfolio

This is the source code for Abdalla Yaser's personal portfolio website, built with React, Tailwind CSS, Framer Motion, and i18next.

## Setup

To run the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abdallayasser17/Abdalla-Yaser-Portfolio.git
   cd Abdalla-Yaser-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

## Deployment

The project is configured for deployment to Vercel. To deploy the website, follow these steps:

1. **Install the Vercel CLI:**
   ```bash
   npm install -g vercel
   ```

2. **Deploy the project:**
   ```bash
   vercel --prod
   ```
   This will deploy the production build of the website to Vercel.

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- i18next (bilingual)
- Three.js + React Three Fiber (for orbs)
- Chart.js (stats)
- Formspree (contact)
- Vercel (deploy)
